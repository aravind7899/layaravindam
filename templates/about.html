Layaravindam

Layaravindam is a Korvai (Mukthai) akshara calculator application. This application consists of default patterns and custom patterns which the user can define.

The default patterns consist of Chaturasram, Trisram, Khandam, Misram based patterns and Chatusram and Trisram combination patterns.

The custom patterns have two options. The first set gives only Korvai (Mukthai) aksharas. The second pattern is defined with Uttarangam and gives a few combinations of Purvangam and Uttarangam aksharas for that particular pattern.

This repo consists of the source code which is built using Python Flask as backend and HTML, CSS, JavaScript as frontend. This application is deployed on Heroku cloud.
